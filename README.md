# Logistics Robot with Computer Vision
QMUL EMS690U Integrated Design Project

Real-time grocery item detection (Apple, Banana, Lemon, Orange) using YOLOv8, integrated with a ROS 2 robot arm for autonomous pick and place.

## Repository Structure
- ros_ws/ - ROS 2 workspace
- computer_vision/ - YOLO model and results  
- notebooks/ - Google Colab training notebooks
- testing/ - Validation results and screenshots
- docs/ - Documentation and report figures
